# Videography ideas

Shoot in split mode which contrast each other.

Day on top/ night on bottom
Where day start sleep to work and in contrast to it night goes from work to sleep.
They go hand in hand basically 0->1 / 1-> 0

Colors should be opposite of the color wheel. -> Warm/Cool or vice versa

Reference : Gawx Art -> Perfect Imperfection.
